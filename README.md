# PortfolioHUB
Plataforma centralizada de portfólio digital, implantada como desafio do Bootcamp I CEUB, com foco em integração GitHub e práticas de segurança.
Planejamento da Implantação.

Etapa,Requisito da Entrega,Status,Recursos Necessários
1. Planejamento,Criar este plano detalhado e configurar o GEMINI (Eu) como guia.,Em Andamento,"Google Docs/Sheets, Gemini."
2. Configuração Git/GitHub,Criar repositório dedicado e integrar o armazenamento de projetos no PortfolioHUB.,Pendente,"GitHub (Repo PortfolioHUB), Google Sites."
3. Gestão de Usuários e Segurança,Configurar a gestão de usuários no GitHub e implementar políticas de segurança.,Pendente,"GitHub, Gemini para guiar as políticas."
4. Compartilhamento/Controle,Integrar o repositório ao PortfolioHUB para controle de versão e colaboração.,Pendente,"Google Sites (Embed/Links), GitHub."
5. Finalização e Testes,Completar todas as integrações e preparar para o lançamento.,Pendente,PortfolioHUB (Google Sites).
6. Revisão e Apresentação,Preparar e entregar a apresentação (YouTube).,Pendente,"Google Apresentações, YouTube."

Ferramentas Utilizadas:

Google GEMINI: É utilizado como ferramenta de apoio e guia durante todo o processo de implantação.

Google Workspace: A suíte de ferramentas do Google (provavelmente incluindo Google Sites, Drive, Docs, etc.) será usada para hospedar a plataforma PortfolioHUB.

Git e GitHub: Serão usados para configurar o ambiente do projeto, gerenciar o controle de versão, o compartilhamento e o controle de acesso, além de servir como armazenamento de projetos.

YouTube: Será utilizado para hospedar a breve apresentação final sobre as etapas da implantação do PortfolioHUB.
